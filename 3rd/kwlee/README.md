# Reaction_rate(반응속도 측정)

- KeyboardEvent API를 사용하여 키보드 입력을 받아 반응속도를 측정해 브라우저에 출력해주는프로그램입니다.

- 총 5번의 입력을 받아 평균 반응속도를 ms단위로 출력해줍니다.
